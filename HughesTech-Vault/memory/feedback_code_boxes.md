---
name: Code boxes for copyable content
description: Always put copyable values (DNS records, commands, keys, config) in code boxes so user can copy easily
type: feedback
---

Always put any copyable content in code boxes — DNS records, terminal commands, API keys, config values, etc.

**Why:** User explicitly requested this as standard protocol. They need to copy-paste values accurately into forms and terminals.

**How to apply:** Any time there's a value the user needs to copy (a command to run, a DNS record to set, a key to paste, a config value), wrap it in a code box. No exceptions.
