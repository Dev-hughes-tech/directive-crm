NEVER use $45, $55, or $65 per user per month for Directive CRM. Those numbers were wrong and have been corrected.

**Official Directive CRM subscription tiers:**
- Basic: $175/month
- Plus: $325/month
- Pro: $575/month
- Enterprise: $1,200/month — includes 4 sales rep accounts + 1 management account

**Why:** User corrected this directly after external critique from Gemini and Grok exposed the wrong pricing in documents being shared externally.

**How to apply:** Any time pricing is mentioned in a document, conversation, or code for Directive CRM, use only these four price points.
